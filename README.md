# Sales retail transactions

## Overview
This project demonstrates both Manual and Automatic Data Cleaning techniques using Python and Pandas.

## Dataset
Sales Retail Transactions Dataset

## Data Cleaning Performed

### Manual Cleaning
- Removed duplicate records
- Handled missing values
- Removed extra spaces
- Standardized text formatting
- Converted date columns
- Checked data types

### Automatic Cleaning
- Automated missing value handling
- Automated duplicate removal
- Automated text cleaning
- Automated date conversion

## Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Output
Cleaned dataset saved as:
- cleaned_sales_data.xlsx
